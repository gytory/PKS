# Двухпанельный файловый менеджер (Far Manager Lite)

Консольный двухпанельный файловый менеджер для Windows. Позволяет копировать, перемещать, удалять и открывать файлы.

## Возможности

- Две независимые панели
- Навигация с клавиатуры
- Копирование (C), перемещение (M), удаление (D) файлов
- Открытие файлов (Enter)
- Подтверждение опасных операций

## Управление

| Клавиша | Действие |
|---------|----------|
| ↑ / ↓ | Навигация по файлам |
| Enter | Войти в папку / открыть файл |
| Backspace | На уровень выше |
| Tab | Переключить панель |
| C | Копировать файл |
| M | Переместить файл |
| D | Удалить файл |
| Q / ESC | Выход |

## Сборка и запуск

### Требования
- Windows 10/11
- Компилятор C++17 (MinGW или Visual Studio)

### Способ 1: Через g++ (MinGW)

```bash
g++ -std=c++17 src/*.cpp -o file_manager.exe -I src

./file_manager.exe
